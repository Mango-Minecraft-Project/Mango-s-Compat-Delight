# FTB 連鎖挖礦（FTB Ultimine）

## 沃土

沃土現在可以使用 ![](@minecraft:item/diamond_hoe)鋤頭 連鎖耕地，使其成為沃土耕地

## 蘑菇菌落

現在可以使用帶有 ![](@minecraft:item/shears)`c:tools/shears` 標籤的物品連鎖收穫蘑菇菌落，並會消耗一個蘑菇用於重新種植。
